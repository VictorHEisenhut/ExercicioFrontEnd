Site desenvolvido para a realização dos exercícos passados pelo professor sobre o conteúdo de HTML, CSS e JavaScript no curso do Entra21 da Proway

  • O 1º exercício se trata deste site com logomarca barra de navegação com as atividades listadas

  • O 2º exercício lê a idade do usuário e retorna se ele é maior de idade ou não

  • O 3º exercício lê 3 notas de um aluno e frequência e retorna se ele foi aprovado ou reprovado de acordo com sua média

  • O 4º exercício pede para o usuário: a sua altura e peso, e então, retorna o IMC correspondente aos dados

  • O 5º exercício serve para efetuar aumentos salariais de acordo com o cargo do usuário
